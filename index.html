Inheritance and Prototype

1.Create 2 objects - parent and child. Attach methods to parent and use those methods in child object using parents prototype

            const parentObj={
                calAge : function(){
                    console.log(`${this.fname} ${this.lname} ${this.age}`)
                }
            }

            const childObj = Object.create(parentObj);
            childObj.fname="ainan";
            childObj.lname="salaha";
            childObj.age="18";
            
            childObj.calAge();

    Output: Ainan Salaha 18

2.Write code to explain prototype chaining

            function Person() {
                this.name = 'Salaha'
            }
            
            // adding property 
            Person.prototype.name = 'Ainan';
            Person.prototype.age = 23
            
            const person1 = new Person();
            
            console.log(person1.name); // Salaha
            console.log(person1.age); // 23

    Output: Salaha
            23
